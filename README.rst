madrasa_s3 plugin for `Tutor <https://docs.tutor.edly.io>`__
############################################################

madrasa_s3 plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/madrasafree/tutor-contrib-madrasa-s3

Usage
*****

.. code-block:: bash

    tutor plugins enable madrasa_s3
